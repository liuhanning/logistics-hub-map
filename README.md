# 国家物流枢纽布局图

可视化展示国家发改委公布的国家物流枢纽布局信息。

## 功能特性

- **地图总览**: 在全国地图上展示所有国家物流枢纽的分布
- **分类筛选**: 按枢纽类型（陆港型、港口型、空港型等）筛选
- **批次筛选**: 按建设批次（2019-2025 年）筛选
- **省份筛选**: 按省份/直辖市筛选
- **统计分析**: 查看类型分布、批次分布、省份 TOP10

## 数据结构

### 枢纽类型
- 陆港型 (53 个)
- 港口型 (34 个)
- 空港型 (23 个)
- 生产服务型 (33 个)
- 商贸服务型 (36 个)
- 陆上边境口岸型 (17 个)

### 建设批次
- 第一批 (2019 年): 23 个城市入选
- 第二批 (2020 年): 30 个城市入选
- 第三批 (2022 年): 20 个城市入选
- 第四批 (2023 年): 部分城市增补
- 第五批 (2024 年): 部分城市增补
- 第六批 (2025 年): 部分城市增补

## 使用方法

### 启动本地服务器

```bash
cd web
python -m http.server 8080
```

然后访问：http://localhost:8080

### 生成/更新数据

```bash
python build_logistics_data.py
```

## 数据来源

- 国家发展改革委
- 国家物流枢纽布局和建设名单 (2019-2025)
