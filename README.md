# 国家物流枢纽布局图

可视化展示国家发改委公布的国家物流枢纽布局信息。

## 功能特性

- **地图总览**: 在全国地图上展示所有国家物流枢纽的分布
- **多类型枢纽聚合显示**: 同一城市有多个类型枢纽时，使用环形堆叠标记展示，点击可查看详情
- **分类筛选**: 按枢纽类型（陆港型、港口型、空港型等）筛选
- **批次筛选**: 按建设批次（2019-2025 年）筛选
- **省份筛选**: 按省份/直辖市筛选
- **统计分析**: 查看类型分布、批次分布、省份 TOP10

## 多类型枢纽显示

当一个城市有多个类型的物流枢纽时（如上海有 4 种类型、南京有 5 种类型），地图会使用**环形堆叠标记**展示：

- **环形标记**: 外环分段显示各种类型的颜色，内圆显示类型数量
- **悬停效果**: 显示城市名称和类型数量
- **点击详情**: 弹出信息窗口，列出该城市所有枢纽类型的详细信息

- 第五批 (2024 年): 部分城市增补
- 第六批 (2025 年 2 月): 部分城市增补

## 使用方法

### 启动本地服务器

```bash
cd web
python -m http.server 8008
```

然后访问：http://localhost:8008

### 生成/更新数据

```bash
python build_logistics_data.py
```

## 数据来源

- 国家发展改革委
- 国家物流枢纽布局和建设名单 (2019-2025)
